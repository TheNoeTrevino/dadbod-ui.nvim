## 1.0.0 (2026-07-20)


### ⚠ BREAKING CHANGES

* drop the :DBUI* user commands
* key->action registry with user-defined actions
* group config options by context
* drop g:db_ui_* legacy global config support
* require Neovim 0.12, adopt vim.fs.abspath and vim.fs.rm
